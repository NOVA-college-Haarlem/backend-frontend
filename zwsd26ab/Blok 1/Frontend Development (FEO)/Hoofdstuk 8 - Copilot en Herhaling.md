## Hoofdstuk 8: Copilot & Herhaling

### Leerdoelen

Na deze les kan de student:

- Uitleggen wat GitHub Copilot is en hoe het werkt
- Copilot verantwoord gebruiken als hulpmiddel
- Eerder geleerde HTML- en CSS-concepten zelfstandig toepassen
- Eigen code beoordelen en verbeteren

### Lesopbouw

**1. Terugblik**

- Projectweek 1 nabespreken
- Wat ging goed? Wat was moeilijk?
- 2–3 studenten laten hun pagina tonen aan de klas

**2. Copilot Introductie**

**Wat is Copilot?**

- AI-assistent voor programmeren, ingebouwd in VS Code
- Suggereert code terwijl je typt, op basis van context en comments
- Kan helpen — maar je moet begrijpen wat je accepteert!

**Installatie:**

- GitHub Copilot extensie in VS Code installeren
- Inloggen met school-account

**Demonstratie:**

- Type een comment: `/* Geef de body een lichtgrijze achtergrond */`
- Laat zien hoe Copilot een suggestie geeft
- Druk Tab om te accepteren, Esc om te weigeren

**Belangrijke regels:**

1. **Begrijp wat je accepteert** — Als je het niet snapt, gebruik het niet
2. **Test altijd** — Kijk in de browser of het werkt zoals bedoeld
3. **Copilot is een hulpmiddel** — Jij blijft de programmeur
4. **Leer eerst zelf** — Nu je de basis kent, mag je Copilot inzetten

**Oefening met Copilot:**

Laat studenten comments typen en Copilot-suggesties beoordelen:

- `/* Maak alle paragrafen grijs */`
- `/* Centreer de h2 koppen */`
- `/* Geef de nav een donkere achtergrond en witte tekst */`

Bespreek klassikaal: begrijp je wat Copilot voorstelt? Is het correct?

**3. Herhaling & Verdieping**

Studenten verbeteren hun eigen hobby-pagina op basis van feedback uit projectweek:

- Controleer de HTML-structuur: zijn alle semantische elementen aanwezig?
- Verbeter de CSS: consistente kleuren, leesbaar lettertype
- Voeg een `.card` class toe met padding en border
- Test met DevTools: kloppen margin en padding?

**Aandachtspunten om klassikaal te herhalen:**

- Verschil `<div>` vs semantische elementen
- Wanneer class, wanneer ID?
- Hoe werkt het box model? (padding binnen, margin buiten)

**4. Afronding**

- Vragen?
- Volgende week: Projectweek 2 — eindproject van Blok 1!

### Opdracht

Verbeter je hobby-pagina vóór de volgende les:

- Minstens 1 verbetering op basis van Copilot-suggestie (vermeld welke)
- Zorg dat de semantische structuur klopt
- Voeg minstens 1 `.card` element toe met border en padding
